Información Importante
Formato de Archivos
  • Solo se aceptan archivos en formato Excel (.xlsx, .xls)
  • La primera fila debe contener los nombres de las columnas
  • Los datos deben comenzar en la fila 2
  • Use el botón "Validar Formato" antes de importar
Tipos de Importación
  • Completa: Procesa múltiples hojas del archivo
  • Solo Grupos: Procesa únicamente la hoja "Grupos"
  • Solo Entregas: Procesa únicamente la hoja "Entregas"
  • Solo Estudiantes: Procesa únicamente la hoja "Estudiantes"
  • Solo Equipos: Procesa únicamente la hoja "Equipos"
  • Consulte la información de formato mostrada arriba
  • El proceso puede tardar unos minutos
Importante: Valide siempre el formato antes de realizar la importación para evitar errores.
